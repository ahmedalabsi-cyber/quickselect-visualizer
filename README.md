# QuickSelect / Order Statistics Visualizer

An interactive step-by-step visualization of the **QuickSelect algorithm** built entirely with vanilla HTML5, CSS3, and JavaScript — no libraries or frameworks.

🔗 **Live Demo:** https://ahmedalabsi-cyber.github.io/quickselect-visualizer/

---

## Features
- **Array Builder** — add, remove, shuffle custom elements or load presets
- **Step-by-step animation** — watch partitioning, pointer movement, and search space reduction in real time
- **Pseudocode panel** — active line highlighting with tooltips
- **Complexity page** — interactive O(n) vs O(n²) growth chart
- **D&C Paradigm page** — recursion tree comparing QuickSelect vs QuickSort
- **Learn page** — worked example, key concepts, and common mistakes

## Algorithm
- Lomuto partition scheme
- Average case: **O(n)**
- Worst case: **O(n²)**
- Space: **O(1)** in-place
